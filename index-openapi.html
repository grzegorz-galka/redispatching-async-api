Zgłoszenie z uOZE przekazanych do OSD poza systemem WOZE

Authorizations:
BearerAuth
Request Body schema: application/json
Array
mRID
string

Unikalny identyfikator MWE

redispatchDate
string <date>

Doba redysponowania wynikająca z polecenia wydanego przez OSP

notificationDate
string <date>

Data kiedy OSD został poinformowany o tym czy i jaka część zredukowanej energii powinna zostać rozliczona w danym systemie wsparcia

supportingSystem
string
Enum: "auction" "auctionWith" "greenCertificate"

Systemie wsparcia w jakim powinna zostać rozliczona energia

energy
integer <int32>

% zredukowanej energii, która powinna zostać rozliczona w danym systemie wsparcia (art. 93 ust. 18 ustawy o odnawialnych źródłach energii (Dz.U.2023.1436))

ipaNumber
string

numer IPA, którego dotyczy system wsparcia
